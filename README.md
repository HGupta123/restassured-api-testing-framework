RestAssured API Testing Framework

A comprehensive Java-based API testing framework using RestAssured, TestNG, and Maven. 
Perfect for learning API automation testing with real-world examples.

Features:
- CRUD operations (GET, POST, PUT, PATCH, DELETE)
- Response extraction and data validation
- Request chaining concepts
- Dynamic payload generation
- Centralized configuration management
- Comprehensive documentation and comments

Getting Started:
1. Clone the repository
2. Run `mvn clean test` to execute all tests
3. Modify BaseTest.java to change API endpoints
